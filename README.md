## console-l

Inspired by parody of the best practices preached by 10x Developer

## Installation

```
npm i console-l
```

## Usage

Import `l` from the package

```
import { l } from 'console-l';
```

Use it anywhere in your code! 

## Examples

```
l('Hello, World!')
```
